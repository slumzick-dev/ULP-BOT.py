# 🔎 Lookup-Bot

**Lookup-Bot** คือบอท Discord สำหรับค้นหา Log ผ่าน DinoDonut API  
ผลลัพธ์จะถูกส่งให้ผู้ใช้งานทาง **DM เท่านั้น** เพื่อความปลอดภัย

> ⚠️ โปรเจ็กต์นี้จัดทำเพื่อการศึกษาและใช้งานตามเงื่อนไขของ API เท่านั้น

---

## ✨ คุณสมบัติหลัก
- 🔍 ค้นหา Log ด้วย keyword
- ⚙️ เลือกโหมดผลลัพธ์ (0 = login:pass, 1 = url:login:pass)
- 📬 ส่งไฟล์ผลลัพธ์ทาง DM อัตโนมัติ
- 📦 แยกไฟล์เมื่อขนาดเกินที่กำหนด
- 🧾 บันทึกประวัติการค้นหา
- 🖱️ UI Button + Modal ใช้งานง่าย
- 💻 ใช้งานง่ายด้วย **ไฟล์เดียว** `ULP-BOT.py`
